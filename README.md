# opengl
### Prerequisite
* Python3
### Run OpenGL on your computer
1. Clone the repo
   ```sh
   git clone https://github.com/Namanskshetty/opengl.git
   ```
2. Go to folder
   ```sh
   cd opengl
   ```
3. Give the executable permission to bin.sh file
   ```sh
   chmod +x bin.sh
   ```
4. Run python file
   ```sh
   python3 main.py
   ```
